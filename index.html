Progress Bar .JS

Create responsive and stylish progress bars for the web!

How it works

  • Progress bars are just regular SVG paths.
  • SVG = scalable vector graphics.
    • They do NOT lose quality if they are zoomed or resized.
    • Every element and every attribute in SVG files can be animated.

Full Examples of progress bars:

  • Minimal
  • File Upload

  • Password Strength

Fancy Vocabulary

  • .svg: reference to SVG element where progress bar is drawn
  • .path: reference to SVG path which presents the actual progress bar
  • .trail: reference to SVG path which presents the trail of the progress bar. Returns null if trail is not defined
  • .text: reference to p element which presents the text label for progress bar. Returns null if text is not defined
  • .animate(progress,[options],[cb]: animates drawing of a shape
  • .set(progress): sets progress instantly without animation. Clears all animations for path
  • .stop: stops animation to its current position
  • .value: returns current shown progress from 0 to 1. This value changes when animation is running
  • .setText(text): Sets text to a given string
  • .destroy: removes SVG element from container and removes all referenes to DOM elements. Destroying is irreversible
